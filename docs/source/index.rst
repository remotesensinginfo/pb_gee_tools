pb_gee_tools documentation
====================================

**pb_gee_tools**  is a Python module with tools for supporting the use of
the `Google Earth Engine <https://earthengine.google.com>`_
`Python API <https://developers.google.com/earth-engine/tutorials/community/intro-to-python-api>`_
and reducing work for repetitive or common tasks.

Check out the `examples <https://github.com/remotesensinginfo/pb_gee_tools/tree/main/examples>`_
section which can be run using `colab <https://colab.google>`_.

Install using pip:

.. code-block:: console

   (.venv) $ pip install pb_gee_tools

.. note::

   This project is under active development.
